Краткое описание программы и инструкция запуска.

Разработанные на языке программирования Java программы server и client являются компонентами CRM системы музея. Сервер представляет собой симулятор программу-сервер, способная работать с портами компьютера и управляющая основными данными системы. Клиент является программой-клиентом, которая в свою очередь может подключиться к серверу и взаимодействовать с ним на протяжении всего его жизненного цикла.

Для корректного запуска программы требуется сперва запустить сервер и только после его успешной инициализации запускать клиента. Сервер стоит запускать только один раз. Клиентов же могут быть несколько. Также возможен и запуск сервера и клиента на разных операционных системах.
При запуске системы требуются некоторые опции:
1. museum.server.Server.java – 59090 5
2. museum.client.Client.java - 59090 127.0.0.1
